# La Cuenta — AI Context

## What this app is
A receipt-style bill splitting mobile app for friends.
Built with React Native, Expo, and TypeScript.

## Stack
- React Native + Expo SDK 55
- TypeScript
- React Navigation (stack)
- expo-linking (for Venmo/Zelle deep links)

## Folder structure
/screens — all app screens
/components — reusable components (coming soon)
/utils — split calculation logic (coming soon)

## Color palette
- Dark green: #2D4A35
- Medium green: #3B6147
- Light green accent: #7DC98A
- Cream/white: #F8F8F6
- White cards: #FFFFFF

## Design style
Receipt/bill aesthetic. Monospace fonts. Dashed tear lines.
Clean and simple — easy to read.

## Current screens
- SplashScreen — logo + login/signup buttons
- LoginScreen — email + password
- HomeScreen — list of group tabs
