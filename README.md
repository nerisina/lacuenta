# La Cuenta 🧾

A receipt-style bill splitting app for friends.
Split shared and personal expenses, see who owes who,
and pay via Venmo or Zelle in one tap.

## Features
- Create groups for any event
- Add shared and personal expenses
- Auto equal split calculator
- See exactly who owes who
- Pay via Venmo or Zelle in one tap
- Receipt-style UI

## Tech Stack
- React Native + Expo
- TypeScript
- React Navigation

## Getting Started
cd lacuenta
npm install
npx expo start

## Screens
- Splash / Login / Sign Up
- Home — your open tabs
- Group — full receipt view
- Settle Up — pay via Venmo or Zelle
